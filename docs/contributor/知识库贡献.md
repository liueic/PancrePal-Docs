# 小X宝知识库维护

## 可能存在的文档类型与清洗

通常我们要处理的文件有 `pdf`、`docx`、`jpg` 等等，这些文档格式都不大统一，为了保证数据结构的统一和知识库的准确性，我们需要转换格式

这里推荐使用 MinerU 来进行文档处理：[MinerU](https://mineru.net/)

直接点击「在线使用」：

![](https://s3.juniortree.com/pic/2025/11/18700b7fb025c0d862acab99358a6a86.webp)

需要先登陆一下：

![](https://s3.juniortree.com/pic/2025/11/845f6fd136c7cfbba9e0164d29c0f5a8.webp)

![](https://s3.juniortree.com/pic/2025/11/7bc1cdd546748b1917bcd7579bb08dfd.webp)

选择「本地上传」，选择你要处理的文档，可以是 `docx`、`pdf`、`jpg` 等等，只要它让你上传，你就大胆传：

![](https://s3.juniortree.com/pic/2025/11/a0b3c2a4a8a41b99146955e3d7733832.webp)

等待解析：

![](https://s3.juniortree.com/pic/2025/11/f4bb3d8ec18b0eaec52a63be5e9f9765.webp)

等待解析结束，这通常取决于你的文档大小和复杂程度，查看解析的结果是否正确：

![](https://s3.juniortree.com/pic/2025/11/453f01d662106ce258b7fab901d668a0.webp)

如果确定没有问题了，就可以进行下载，右上角有一个下载的图标，选择 Markdown 即可：

![](https://s3.juniortree.com/pic/2025/11/539b4632094b567722dc6cd2f06076f8.webp)

## 导入知识库

进入到后台后，点击左侧的「知识库」

![](https://s3.juniortree.com/pic/2025/11/892e425d070f41f5f4c7c999aeddff30.webp)

请务必选择你要导入数据对应的数据库，如果你不知道选择哪一个，你可以都点进去看一看，我这里要导入的是和复旦大学胰腺门诊相关的数据，因此我选择「复旦肿瘤专用库」，可以看到当前的知识库里面已经存在的类似内容

![](https://s3.juniortree.com/pic/2025/11/75f53b0640a758dbefd193a33a7e626c.webp)

我们点击 `新建/导入`，选择文本数据集

![](https://s3.juniortree.com/pic/2025/11/050fd204f887baf5c1be8fe2c9548e2f.webp)

这里选择上传文件

![](https://s3.juniortree.com/pic/2025/11/c148b1897448bb2afa88f07983ff9110.webp)

选择我们刚才处理好的文档，进行上传，按照下面的进度跑到 100% 即为上传完成

按照默认设置即可，结束知识库导入

![](https://s3.juniortree.com/pic/2025/11/e3f34f50f5ca441d6bc73a9ea79e6d40.webp)

之后回到知识库页，如果觉得这个文件名不太好区分，可以进行重命名：

![](https://s3.juniortree.com/pic/2025/11/1321b72f45c86589bff29598a0844ff9.webp)

![](https://s3.juniortree.com/pic/2025/11/ce9c3a52c3dc8d8e6bc40e6988fb8ab2.webp)
